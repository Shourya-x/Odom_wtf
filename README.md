odom_wtf started as a simple question: "Can I build a robot from scratch and actually understand every piece of the ROS pipeline?"

The answer is maybe. The journey is documented here.

This rover covers:

    Embedded motor control (Arduino/STM or direct)

    Sensor fusion & odometry

    ROS nodes, topics, services, actions

    Mapping, localization, path planning

    Basic computer vision (ideally)

Expect messy code, triumphant commits, and the occasional segfault.

Status: Learning in public. PRs or advice welcome. Laughter at my tf struggles also welcome.
