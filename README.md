# odom_wtf

> *"Where your robot thinks it is, and where your sanity isn't."*

`odom_wtf` started as a simple question:

**"Can I build a robot from scratch and actually understand every piece of the ROS pipeline?"**

The answer is *maybe*. The journey is documented here.

---

## What this rover covers

- Embedded motor control (Arduino/STM or direct)
- Sensor fusion & odometry
- ROS nodes, topics, services, actions
- Mapping, localization, path planning
- Basic computer vision (ideally)
